Crops the image.

Parameters
imgThe image that is being cropped.
leftThe number of percentages by which the image will be cropped from left.
topThe number of percentages by which the image will be cropped from top.
rightThe number of percentages by which the image will be cropped from right.
bottomThe number of percentages by which the image will be cropped from bottom.

This function does not perform any range-checking for left, top, right, bottom. The function cuts the picture on each side depending on the parameters left, top, right, bottom. For instance, left means that left pixels are cut from the left.

Inserts one image into another.

Parameters
imgThe image on which the other image will be inserted.
otherThe image that will be inserted on img image.
xThe x-coordinate on which the left-top upper pixel of the other image will be inserted.
yThe y-coordinate on which the left-top upper pixel of the other image will be inserted.

The function insert other image on the img image starting from the pixel with the x and y coordinates. Note, if the x and y parameters are selected in such a way that the other image does not fall on the img image, then there will be no changes in the img image.
